# API Testing – Restful Booker API

A comprehensive **Manual API Testing** project for the **Restful Booker REST API**, demonstrating an end-to-end Software Testing Life Cycle (STLC) with industry-standard QA documentation and Postman.

## Project Objective

- Validate REST API functionality
- Perform positive & negative testing
- Verify authentication & authorization
- Ensure complete requirement traceability
- Report and track defects professionally

## Tech Stack
